### 普通文本
直接输入的文字就是普通文本。
需要注意的是要换行的时候不能直接通过回车来换行，需要使用\<br>(或者\<br/>)。也就是html里面的标签。事实上，markdown支持一些html标签，你可以试试。当     然如果你完全使用html来写的话，就丧失意义了，毕竟markdown并非专门做前端的，然而仅实现一般效果的话，它会比html写起来要简洁得多得多啦。<br/>

这是一段普通的文本,
直接回车不能换行，<br>要使用\<br>
